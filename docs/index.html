Another Obscure Coding Language

This is an online interpreter for AOC Lang created by me. See more about syntax on the aoc-lang github repo.

isPrime = fn(n) {
    m = 2
    prime = 1
    while m * m <= n {
        if n % m == 0
            prime = 0
        m = m + 1
    }
    prime
}

n = 2
while n < 100 {
    if isPrime(n)
        print n
    n = n + 1
}
